Use the **trend-radar** skill to produce a time-stamped scan of what's currently working in marketing.

Lens: **$ARGUMENTS**

Default window: last 90 days. User can override.

Apply the "real signal" filter — every trend needs a named brand example launched in-window with a credible source.

End with the "what I deliberately left out" section and the next-refresh date.
